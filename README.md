# Telegram бот с мудрыми цитатами

Telegram-бот, который выдает случайные картинки с цитатами из Яндекс Диска.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` на основе `.env.example` и заполните его:
   - `TELEGRAM_BOT_TOKEN` - токен вашего Telegram бота (получить у [@BotFather](https://t.me/BotFather))
   - `YANDEX_DISK_TOKEN` - OAuth токен Яндекс Диска (получить на [oauth.yandex.ru](https://oauth.yandex.ru/))
   - `YANDEX_DISK_FOLDER` - путь к папке с картинками (по умолчанию `/Wisdom_Cards`)

## Получение токенов

### Telegram Bot Token
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен
4. Скопируйте токен в `.env` файл

### Яндекс Диск OAuth Token
1. Перейдите на [oauth.yandex.ru](https://oauth.yandex.ru/)
2. Создайте новое приложение
3. Получите OAuth токен
4. Скопируйте токен в `.env` файл

## Подготовка Яндекс Диска

1. Создайте папку на Яндекс Диске (например, `Wisdom_Cards`)
2. Загрузите туда картинки с цитатами (поддерживаются форматы: jpg, jpeg, png, gif, bmp, webp)
3. Укажите путь к этой папке в переменной `YANDEX_DISK_FOLDER` в файле `.env`

## Запуск

```bash
python bot.py
```

## Использование

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Нажмите кнопку "Получить подсказку"
4. Бот отправит случайную картинку с цитатой

## Размещение на Bothost

Для размещения бота на хостинге Bothost (чтобы он работал 24/7) см. подробные инструкции в файле [DEPLOYMENT.md](DEPLOYMENT.md).

**Краткая инструкция:**
1. Зарегистрируйтесь на [bothost.ru](https://bothost.ru/)
2. Создайте проект и подключите Git репозиторий
3. Добавьте переменные окружения (токены)
4. Разверните бота
5. Готово! Бот будет работать 24/7 на бесплатном тарифе

## Структура проекта

```
Wisdom_Card/
├── bot.py              # Основной файл бота
├── yandex_disk.py      # Модуль для работы с Яндекс Диском

├── test_connection.py  # Скрипт для проверки подключения

├── .gitignore          # Игнорируемые файлы (config.py защищен!)
├── DEPLOYMENT.md       # Инструкции по развертыванию на Bothost
├── images/
│   └── sleep-cat.png   # Картинка для сообщения о "сне" бота
└── README.md           # Документация
```

**Важно:** Файл `config.py` с токенами **НЕ попадет** в Git благодаря `.gitignore`!
